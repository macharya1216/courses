Introduction to Propositional Logic

Propositional logic is the foundation of logical reasoning. It deals with propositions (statements that are either true or false) and how they can be combined using logical operators.

What is a Proposition?

A proposition is a declarative sentence that is either true or false, but not both.

Examples of propositions:

  • "It is raining" (P)
  • "2 + 2 = 4" (Q)
  • "The sky is blue" (R)
